Bambu Status Bar

Flash the firmware to your ESP32 straight from this page — no PlatformIO, no Arduino IDE, no config files to edit.

1. Connect your ESP32

Plug the ESP32 into this computer with a USB cable (the same one you'll use to power it from the printer works fine).

2. Install

Your browser doesn't support Web Serial. Open this page in Chrome or Edge on a desktop computer instead — Web Serial isn't available in Firefox, Safari, or on mobile. This page needs to be served over HTTPS (or localhost) to access USB devices. If you're seeing this on the published site, please open an issue. Works in Chrome / Edge on desktop.

3. Finish setup over WiFi

Once flashing completes, the ESP32 reboots and opens its own WiFi network to configure itself — no cables needed for this part:

  1. On your phone or laptop, join the WiFi network named BambuStatusBar-Setup.
  2. A setup page should open automatically (a "captive portal"). If it doesn't, open a browser and go to 192.168.4.1.
  3. Enter your home WiFi details, plus your printer's IP address, serial number, and LAN access code, then save.

Not sure where to find your printer's IP/serial/access code, or the setup page isn't showing up? See the troubleshooting guide.
